Steps to the Raspberry pi working
---------------------------------

Step 1 - power on the raspberry pi
Step 2 - create the following hotspot name and password then it will auto connect to the wifi
=============================================================
Network name ==Home,
Network Password == 1234567890
=============================================================
step 3 - connect the ssh using the ip address with following user name and password
======================================================================
Username= pi
password=raspberry
==========================================================================

step 4 - run the command "bash run.sh" #for the normal run it will send the file to the system
step 5 - run the command "bash run_model.sh" for the run with loaded model in raspberry



Steps for the system server 
--------------------------

Step 1 - open this folder in the vscode
Step 2 - Install the requirements using the command "pip install -r requirements.txt"
Step 3 - To run the server code use the command or run the main.py file "python main.py"
